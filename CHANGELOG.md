# 更新日志

## [1.3.0] - 2026-05-06

### 新增
- ✨ 新增 `display_mode` 显示模式配置
  - `vertical`（默认）：纵向列表，依次显示所有设备
  - `horizontal`：横向三列布局，按状态分组（正常 | 预警 | 紧急）
- ✅ `sort_order` 排序顺序确认支持
  - `asc`：按电量升序（低电量 → 高电量）
  - `desc`：按电量倒序（高电量 → 低电量）

## [1.1.0] - 2026-05-06

### 新增
- ✅ 新增 `entities` 配置项，支持手动指定要显示的实体列表
- ✅ 新增 `device_class: battery` 自动识别

### 修复
- 🐛 修复实体识别过于宽泛的问题
  - 之前：所有 state 为 0-100 的传感器都被当作电池
  - 现在：必须满足以下任一条件才会被识别：
    1. `device_class` 为 `battery`
    2. 实体 ID 或 friendly_name 包含电池相关关键词（battery、电量、电池等）

## [1.0.0] - 2026-05-06

### 新增
- 🎉 首次发布
- 🔋 自动扫描 Home Assistant 中所有电池实体
- 📊 显示电池电量百分比和状态图标
- 🎨 进度条可视化电池使用情况
- 🚦 三色状态指示（正常/预警/紧急）
- 📱 支持按房间/区域分组显示
- 🔄 支持按电量/名称排序
- ⚙️ 支持自定义阈值（预警/紧急）
- 🎨 支持自定义颜色主题
- ✨ 浅色主题设计，清晰易读
